## 一、分類體系盤點

### 1. 學術系統地理學（通論地理）的標準切法

[維基百科：地理學](https://zh.wikipedia.org/zh-tw/%E5%9C%B0%E7%90%86%E5%AD%A6)、[自然地理學](https://zh.wikipedia.org/zh-tw/%E8%87%AA%E7%84%B6%E5%9C%B0%E7%90%86%E5%AD%B8)：

- **自然地理**五大要素：**地形、氣候、水文、土壤、生物**（大學普地教科書的通用骨架，對應岩石圈／大氣圈／水圈／土壤圈／生物圈）。
- **人文地理**五大要素：**人口、聚落、經濟（產業）、交通、政治**；再加文化（宗教、語言、歷史地理）。

### 2. 108 課綱（國中臺灣地理）的切法

[社會領域課綱](https://cirn.moe.edu.tw/Upload/file/27799/68319.pdf)三大主題「基本概念與臺灣／區域特色／地理議題」，落到七年級教科書（[康軒七上](https://www.learnmode.net/course/518321/content)）就是元素序：
**位置 → 地形 → 海岸 → 氣候 → 水文 → 生態環境**（上學期），**人口 → 族群 → 聚落 → 交通 → 產業 → 環境議題**（下學期）。值得注意：課綱每個項目都配「問題探究」條目——官方也承認元素要用問題帶，不是名詞條列。

### 3. 遠足文化《台灣地理百科》100 冊——最值得偷師的前例

（[書系總覽](https://book.douban.com/series/1436)、[套書頁](https://www.shop2000.com.tw/8389/product/p21741517)）
臺灣第一套以「地理」為軸的百科，切法完全是主題式小書：《台灣的溫泉》《台灣的斷層》《台灣的土壤》《台灣的河流》《台灣的氣候》《台灣的濕地》《台灣的老街》《台灣的燈塔》《台灣的古地圖》《台灣的原住民》《台灣的老火車站》《台灣的行政區變遷》《台灣的咖啡》……自然／人文／產業三路並進。**「臺灣的Ｘ」這個句式親切、可無限擴充、書店驗證過 20 年**——我們的主題頁命名直接繼承它。

### 4. 華德福 5–8 年級地理取徑（對照組）

（[waldorf-resources 垂直課綱](https://www.waldorf-resources.org/en/vertical-curriculum/geography)、[Christopherus](https://www.christopherushomeschool.com/the-christopherus-curriculum-geography-through-the-grades/)、[A Waldorf Journey](https://www.awaldorfjourney.com/2023/07/geography-in-the-waldorf-school/)）

- G4 家鄉 → **G5 自己的國家（分區＋人與土地的經濟活動）** → G6 地球科學（地質、氣候）＋更大區域 → G7–8 世界文化地理。
- 核心：**人在中心**——地理是「人與大地的關係」，經濟活動被理解為「人作為自然的夥伴」；方法是現象學式（先觀察描述現象，再追問成因條件）。
- 對本案的啟示：主題頁不能停在自然成因，**每一頁的終點都必須落在「人因此怎麼生活」**——這正好就是 David 給的土壤黑紅範例的結構，也與縣市頁既有寫法（現象→成因→影響）同構。

## 二、三個 IA 方案比較

### 方案 A：教科書系統式

自然（地形／氣候／水文／土壤／生物）＋人文（人口／聚落／產業／交通／文化），中間層即學科名。

- 優點：教師秒懂、與課綱對表容易、擴充位子明確。
- 缺點：命名冷；每格都「該有」，容易逼出湊數頁；和本庫的敘事氣質不合。

### 方案 B：元素敘事式（推薦）

頂層沿用「自然地理／人文地理」，中間層改成 8 個**問句式主題群**，主題頁標題用「現象鉤子——臺灣的Ｘ」：

```
認識臺灣
├─ 縣市（既有軸：22 縣市＋宜蘭 12 鄉鎮＋臺灣總覽）
│
├─ 自然地理 —— 島的身世
│   ├─ 大地的骨架        地形與地質：島為什麼長這樣
│   │     ├─ 還在長高的島——臺灣的山脈
│   │     ├─ 大地會動——臺灣的地震與斷層
│   │     ├─ 地底的餘溫——臺灣的溫泉與地熱
│   │     └─ 島的四種邊——臺灣的海岸
│   ├─ 天空的脾氣        氣候：風和雨從哪裡來
│   │     ├─ 雨下在哪裡——臺灣的雨
│   │     ├─ 風從海上來——臺灣的風
│   │     └─ 每年來訪的巨風——臺灣的颱風
│   ├─ 水的旅行          水文：水從山到海走過的路
│   │     ├─ 急性子的河——臺灣的河流
│   │     └─ 湧出來的水——臺灣的湧泉、湖泊與地下水
│   └─ 土壤與生靈        土壤與生態：腳下的顏色、身邊的生命
│         ├─ 黑土與紅土——臺灣的土壤
│         └─ 一座島、好幾層世界——臺灣的垂直生態
│
└─ 人文地理 —— 人的足跡
    ├─ 人從哪裡來        人口與族群：一波波上岸的人
    │     ├─ 一波波上岸的人——臺灣的移民與族群
    │     └─ 西擠東疏——臺灣的人口分布
    ├─ 人住在哪裡        聚落：地名、庄頭與信仰
    │     ├─ 地名裡的祖先——臺灣的地名
    │     └─ 廟在哪裡，庄就在哪裡——臺灣的廟宇與聚落
    ├─ 人靠什麼生活      產業：土地給的生計
    │     ├─ 把水引到田裡——臺灣的水圳與稻米
    │     ├─ 越高越好喝？——臺灣的茶
    │     └─ 靠海吃海——臺灣的漁與鹽
    └─ 路怎麼連起來      交通：路打開的地方
          └─ 路打開的地方——臺灣的鐵路與公路
```

（樹是完整藍圖；第一批只做 6 頁，其餘格子留空不建假頁。）

- 優點：頂層與縣市頁節名**同名對齊**——主題頁＝一個元素橫看全島、縣市頁＝一個地方縱看所有元素，天然成矩陣；中間層問句就是教學語言；「臺灣的Ｘ」句式有《台灣地理百科》20 年的親切度背書；每群 1–4 頁、可控可擴充。
- 缺點：中間層群名需要 David 認可品味；比方案 A 多一層（但每群頁數少，導覽仍淺）。

### 方案 C：純問題式（不分自然人文）

頂層直接五、六個大問題：「島是怎麼長出來的」「水往哪裡去」「風與雨」「人住在哪裡」「人靠什麼生活」。

- 優點：最徹底的現象學，自然人文在同一問題裡交融（最接近主課程的講述方式）。
- 缺點：與縣市頁的「自然地理／人文地理」節名對不上，交叉連結的落點模糊；擴充到 20+ 主題後大問題會爆掉；教師檢索（「我要找氣候素材」）反而繞路。

**裁決：B。**A 的骨架其實藏在 B 的中間層裡（大地＝地形地質、天空＝氣候、水＝水文……），等於「學科骨、敘事皮」；C 的精神則下放到每一頁的鉤子與正文，不必犧牲導覽結構。

## 四、命名建議（兩大分頁）

| 選項 | 例 | 評估 |
|---|---|---|
| 直用學名 | 自然地理／人文地理 | 教師直覺、可對表課綱；**且縣市頁正文的兩個節名就叫這個**——主題軸沿用同名，矩陣（主題×地方）才對得整齊。缺點是冷。 |
| 全換暖名 | 島的身世／人的足跡（或 山海之島／島上人家） | 有邀請感，但和縣市頁節名脫鉤、檢索不直覺，且「人文」素材放進「人的足跡」哪一格會開始猶豫。 |
| **主名＋副標（推薦）** | **自然地理——島的身世／人文地理——人的足跡** | 導覽與資料結構用學名（穩定、可對齊、可擴充），溫度放在副標與中間層問句（大地的骨架／天空的脾氣／水的旅行／人從哪裡來……）。冷暖各得其所。 |

一句話：**骨架用學名，血肉用暖語**——命名的溫度應該花在中間層與每頁標題（「臺灣的Ｘ」句式），不必動搖頂層的秩序。

## 五、與縣市頁的交叉連結機制

原則：**連結關係單一源頭（SSOT）放在主題頁 frontmatter，縣市母本零改動**，全部由 build script 生成雙向連結。

```yaml
# content/themes/soil.md frontmatter 新增段
theme_group: 土壤與生靈          # 中間層群
regions:                        # 這個現象在哪裡看得到
  - id: yilan                   # 對應既有縣市/鄉鎮 id
    hook: 蘭陽平原的黑色水田土——多雨低地的有機質
  - id: new-taipei
    hook: 林口台地的紅土——被雨淋洗千年的老台地
  - id: yunlin
    hook: 濁水溪沖積扇的「濁水米」土
```

Build 時：

1. **主題頁 → 縣市頁**：`regions` 渲染成頁尾「這個現象在哪裡看得到」連結塊（地名＋一句話 hook，點過去落到該縣市頁的自然／人文地理節）。
2. **縣市頁 → 主題頁**：反向索引自動在縣市頁注入「延伸主題」chips（如宜蘭頁自動長出「臺灣的土壤｜臺灣的溫泉與地熱｜臺灣的雨」）——不必回頭編輯 22＋12 份既有母本。
3. 未來若某縣市頁想手動加連（如南澳→鐵路公路主題），縣市 frontmatter 開選填欄 `related_themes:`，與反向索引合併去重。

主題頁正文結構沿用 CONTENT-SPEC 精神，微調為：**現象開場（鉤子）→ 成因 → 全島分布的差異 → 人的生活方式 → 教學特點（探究問題／跨科連結／五年級即用）→ 說書稿切分提示**。字數建議 2,000 起（比鄉鎮頁略短，因為落地細節交給縣市頁承接）。
