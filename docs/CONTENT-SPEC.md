## 正文結構（正文 2,500 字以上）

1. `## 定位速覽` — ≤150 字：位置、地理骨架一句話、這個鄉鎮最重要的一個教學價值。
2. `## 自然地理` — 地形與成因、水文、氣候、生態。寫「現象→成因→影響」的因果鏈，不是名詞條列。
3. `## 人文地理` — 地名由來、族群與拓墾的歷史層理、聚落分布的邏輯、產業（過去→現在），並回扣自然地理的因果（為什麼人在這裡、這樣生活）。
4. `## 教學特點` — **本資料庫的核心，佔全文 1/3 以上**：
   - `### 探究問題`：3–5 個，現象學觀察式（從學生可觀察、可感受的現象出發推向成因），禁記憶性問答。
   - `### 跨科連結`：這個鄉鎮的素材可以接哪些科目／主題，各一句話說明怎麼接。
   - `### 五年級即用`：可直接用於五年級「臺灣地理」「臺灣歷史」主課程的素材與用法，每條行首標 `【五年級地理】` 或 `【五年級歷史】`。沒有就寫「（本區無五年級即用素材）」。
5. `## 說書稿切分提示` — 標出正文中 2–3 段適合改寫成「學生說書稿」的段落，每段一行 AI 改寫提示，格式：

   > 🗣 切分段落：〈人文地理〉噶瑪蘭人段。AI 提示：改寫為一位噶瑪蘭少年的第一人稱視角，約 5 分鐘口說長度，保留地名與因果，語氣是說故事不是報告。

## 紀律

- 每個事實可追溯：sources 逐條列 URL；查不到就寫「待查證」，不硬掰。
- 段落敘事體，教師可直接讀；避免 bullet 堆砌正文。
- 禁止地方誌流水帳（歷任首長、機關學校列表、節慶羅列）。
- 圖片檢索：TCMB 本機索引 `python3 ~/MyWork/WaldorfTeacherOS-Repo/setup/scripts/tcmb-search.py --search "冬山"`（人文優先）；
  Wikimedia Commons API（自然地理優先），授權作者從 `prop=imageinfo&iiprop=extmetadata` 讀出。
- 縣市級頁面（北北基）同結構，正文 1,500 字以上即可，視野放在全市尺度（不逐區展開）。
