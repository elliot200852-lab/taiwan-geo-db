# 認識臺灣 — 臺灣人文與自然地理資料庫

教師備課用的臺灣地理資料庫：互動地圖導航（臺灣→縣市→鄉鎮），每區以「人文地理／自然地理」雙區塊呈現，
內容以教學價值優先（探究問題、跨科連結、五年級即用標示、學生說書稿切分提示）。

- 第一期範圍：宜蘭縣（鄉鎮級 12 區）＋臺北市、新北市、基隆市（縣市級）
- 母本：`content/{county}/{unit}.md`（撰寫規範見 `docs/CONTENT-SPEC.md`）
- 網站：靜態站（Leaflet＋TopoJSON），GitHub Pages 部署，最終以 link-out app 掛載 CreatorHub
- 規劃沿革：`docs/PLAN.md`

資料與圖片來源均為政府開放資料（OGDL）與 CC 授權素材，頁面自動顯名標示。
